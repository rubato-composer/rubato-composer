Reformer

A reformer allows the conversion of a denotator of one given form to a denotator of another given form, provided that both forms are compatible.

To create a reformer from a form from to a form to, use the static make method of the class Reformer: 

      Reformer reformer = Reformer.make(from, to);

If the reformer can not be created, make returns null.

Then, to reform the denotator d of form from: 

      Denotator res = reformer.reform(d);

The reform throws a RubatoException, if d does not have the required form.
